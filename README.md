# LLMs Course (Spring 2025)

This repository contains the first assignment of the **Large Language Models (LLMs)** course, offered at **University of Tehran** by **Dr. Yaghoubzadeh** and **Dr. Dousti**.

## About the Course
This repository contains projects completed as part of the Large Language Models (LLMs) course at the University of Tehran (Spring 2025).

## CA1
**Objective:**
In this assignment, students are required to analyze a pre-trained language model and evaluate its text generation and response capabilities. The assignment includes the following steps:

1. Load and run an LLM using a suitable platform.
2. Evaluate the model’s performance in text generation and answering specific questions.
3. Analyze the results and compare the model's output against text quality evaluation metrics.
